DNS Lookups

Forward and Reverse DNS lookups are performed to see, if the name to IP and IP to name DNS lookups produce the same results. This feature is used to see if DNS is correctly set up for a host and can be an indicator for a malicious host. Fwd/Rev DNS Match

Email Volume

Email volume uses a log scale with a base of 10. A volume of 10 equates to 100% of the world's email volume. A one point decrease in magnitude equates to a 10x decrease in actual volume. This only gives insight about the total email volume and not about the reputation. Read More. Last Day Vol.

Daily Email Volume
 
 
 
 
 
 Similar to the Richter scale used to measure earthquakes, the Talos Reputation Center volume magnitude is a measure of message volume calculated using a log scale with a base of 10. 
 
 This table illustrates the percentage of internet email associated with each volume magnitude: 
 
 
 
 
 10.0 100% 
 
 9.0 10% 
 
 8.0 1% 
 
 7.0 0.1% 
 
 6.0 0.01% 
 
 5.0 0.001% 
 
 4.0 0.0001% 
 
 3.0 0.00001% 
 
 2.0 0.000001% 
 
 1.0 0.0000001%
